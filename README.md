# GHOST-PROTOCOL-Elite-Autonomous-Cyber-Defense-System
Intercepta e mitiga ameaças em real-time usando IA (Gemini 2.0). Arquitetura poliglota: Gateway em Node.js, Security Brain em Python/FastAPI e Dashboard cyberpunk em Next.js. Inclui banimento automático de IP via Redis, logs em PostgreSQL e orquestração completa com Docker Compose.
